# Integrations

Connect LinqMe to your favorite tools. All integrations are managed from Settings under the Integrations tab.

## Cloud Storage

Connect cloud storage providers to automatically route file uploads from your forms to the right place.

| Provider | Description |
| --- | --- |
| Google Drive | Authenticate with Google and select a destination folder |
| Dropbox | Connect your Dropbox account for file routing |
| OneDrive | Connect Microsoft OneDrive |
| Box | Connect Box.com for enterprise file management |

Once connected, you can set a cloud storage destination on any file upload field in the form builder. When a client uploads a file, it will be automatically sent to your chosen provider and folder.

## Payment Providers

Collect payments directly within your forms by connecting a payment provider:

| Provider | Description |
| --- | --- |
| Stripe | Connect your Stripe account to accept credit card payments |
| PayPal | Connect PayPal via OAuth for PayPal checkout |
| Square | Connect Square for in-form payment processing |

> [!NOTE]
> You must have an active account with the payment provider before connecting. LinqMe does not handle funds directly. All payments go straight to your connected account.

## AI Services

Some advanced field types like the Competitor Analyzer use AI to generate content. Connect an AI provider to unlock these features:

| Provider | Description |
| --- | --- |
| OpenAI | Connect with your OpenAI API key for GPT-powered analysis |
| Anthropic | Connect with your Anthropic API key for Claude-powered features |

Go to Settings, then the Integrations tab, and add your API key under AI Integrations. You can select which model to use from the available options.

## Bot Protection (CAPTCHA)

Protect your forms from spam and automated submissions with CAPTCHA:

| Provider | Description |
| --- | --- |
| reCAPTCHA v3 | Google's invisible bot detection. Runs in the background with no user interaction required |
| Cloudflare Turnstile | Privacy-friendly CAPTCHA alternative. Can be visible or invisible |

Add your site key in Settings under Integrations, then add a Captcha field to any form.
